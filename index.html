Projects

DawnOps

DawnOps is an Application designed for my streaming community. This is a single page application built with React. This application has two types of users an admin and a regular user type. The Application allows a user to use a nave bar to navigate between the Bio, Clips, Games most played , Chat, and Live sections. When a user clicks Live a new tab takes them to my Twitch channel. When the user clicks Clips they are able to watch the clips posted there along with make comments on the clip. When the user clicks on Chat they can see all the messages made by other users and see the name of the user who made the message. They themselves can make their own message. If the user is an admin they will be able to delete any of the Videos, Comments made in the chat section, and any of the games. That being said they are also able to add any video and add any game along with making their own comments in the chat section.

Holiday Road

I built this project with the help of three other teammates. Our project is a web application that allows people to build itineraries for their trips to the beautiful national parks in the United States. The starting point of each trip is Nashville, TN, but the destination will be a national park selected by the user. The user can select a state from a drop down, and then select a national park within that state. Once a park is selected, a 5-day weather forecast is displayed for that area, provided from an external weather API. Next, the user can select an eatery that they would like to stop at along the way, as well as an attraction. Once the itinerary form is fully filled out, the user then can save the itinerary for viewing in the future.
